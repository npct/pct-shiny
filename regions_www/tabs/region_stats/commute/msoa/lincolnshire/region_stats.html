Key information for Lincolnshire, commute layer, MSOA geography

This document provides information about the data underlying the Propensity to Cycle Tool (PCT) commuting layer for Lincolnshire. The data were generated on 2019-11-06 and this document was created on 2019-11-06. The PCT is an open source tool for sustainable transport planning, released under the conditions of the Affero GPL. The PCT source code can be modified by others as long as attribution is made to the original.

This layer of the PCT uses origin-destination data on travel to work from the 2011 Census. The interface is currently displaying the number of commuters travelling by different modes between Middle Super Output Area zones (MSOAs, average size 3,325 commuters). There were 296,400 commuters living in Lincolnshire recorded in the 2011 Census. All of these commuters are included in the zones data that are visualised on the map interface and available for download.

Adding in inter-regional flows, there were 309,212 commuters living or working in Lincolnshire in the 2011 Census. All these commuters are included in the All flows files on the data download pages.

Straight lines, Fast routes and Quieter routes are both visualised on the map interface and available to download. In the downloads, these lines and routes contain all 717 flows between MSOAs that a) start or end in Lincolnshire, b) have a fast route distance <20km, and c) contain at least 10 commuters (by any mode, in either direction). These 717 between-zone MSOA flows contain 145,769 commuters who either live or work in Lincolnshire. To speed up load times, the online map only shows those line and routes with a fast route distance <10km.

Between-zone MSOA flows exclude within-MSOA travel, when the MSOA zone of origin is the same as the zone of destination. Within-MSOA travel is represented by red points on the map when the lines are shown, and accounts for 48,220 commuters living in Lincolnshire. Between-zone flows also exclude those with no fixed place of work.

Finally, the Route Network (LSOA, clickable) and the Route Network (LSOA, image) are built from all between-zone LSOA fast routes (including any that start or end in Lincolnshire). The online version of the clickable route network and the downloads both include all route segments. For more details please see User Manual C1.

See Lovelace et al. (2017) for details of the methods used to estimate the cycling, health and carbon impacts of each scenario, and the methods used to visualise results at the area, line, route and route network level.

Cycling levels in each scenario for Lincolnshire

At baseline (2011 Census), the proportion of commuters who cycled to work in Lincolnshire was 4.4%, compared to the national average of 3.1% in England and Wales as a whole. The percentage of cycling expected was 3.7%, based on the distance and hilliness of commutes in the region using the baseline model for England and Wales.

Table 1 summarises the proportion of commuters cycling, walking, driving and using other modes under each scenario in Lincolnshire, based on aggregating across all residents in all the MSOA zones.

Health and carbon impacts in Lincolnshire

A modified version of the 2014 HEAT tool was used to calculate health impacts, based on the extra physical activity gained through new cycling trips in each scenario, minus the physical activity displaced in any former walking trips that have now been switched to cycling. CO2 impacts are derived from changes in distance driven per scenario.

Table 2 illustrates the health and carbon impacts of the different scenarios in Lincolnshire, based on aggregating across all residents in all the MSOA zones. For each scenario, comparisons are made with a counterfactual in which nobody in the region cycled ('No cyclists'), and with the observed level of cycling in the 2011 Census.
