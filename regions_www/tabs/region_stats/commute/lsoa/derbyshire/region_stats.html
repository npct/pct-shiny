Key information for Derbyshire, commute layer, LSOA geography

This document provides information about the data underlying the Propensity to Cycle Tool (PCT) commuting layer for Derbyshire. The data were generated on 2019-01-11 and this document was created on 2019-01-11. The PCT is an open source tool for sustainable transport planning, released under the conditions of the Affero GPL. The PCT source code can be modified by others as long as attribution is made to the original.

This layer of the PCT uses origin-destination data on travel to work from the 2011 Census. The interface is currently displaying the number of commuters travelling by different modes between Lower Super Output Area zones (LSOAs, average size 690 commuters). There were 439,614 commuters living in Derbyshire recorded in the 2011 Census. All of these commuters are included in the zones data that are visualised on the map interface and available for download.

Adding in inter-regional flows, there were 466,143 commuters living or working in Derbyshire in the 2011 Census, and commuting in a flow containing at least 3 commuters (in either direction). These commuters are included in the All flows files on the data download pages. Flows containing fewer than 3 commuters are excluded from the visualisation and download files for data protection reasons.

Straight lines, Fast routes and Quieter routes are both visualised on the map interface and available to download. These lines and routes contain all 17,611 flows between LSOAs that a) start or end in Derbyshire, b) have a fast route distance <20km, and c) contain at least 5 commuters (by any mode, in either direction). These 17,611 between-zone LSOA flows contain 239,823 commuters who either live or work in Derbyshire.

Between-zone LSOA flows exclude within-LSOA travel, when the LSOA zone of origin is the same as the zone of destination. Within-LSOA travel is represented by red points on the map when the lines are shown, and accounts for 17,088 commuters living in Derbyshire. Between-zone flows also exclude those with no fixed place of work.

The Route Network (LSOA) is built from all between-zone LSOA fast routes (including any that start or end in Derbyshire) that have a fast route distance <20km. For more details please see User Manual C.

See Lovelace et al. (2017) for details of the methods used to estimate the cycling, health and carbon impacts of each scenario, and the methods used to visualise results at the area, line, route and route network level.

Cycling levels in each scenario for Derbyshire

At baseline (2011 Census), the proportion of commuters who cycled to work in Derbyshire was 2.2%, compared to the national average of 3.1% in England and Wales as a whole. The percentage of cycling expected was 2.3%, based on the distance and hilliness of commutes in the region using the baseline model for England and Wales.

Table 1 summarises the proportion of commuters cycling, walking, driving and using other modes under each scenario in Derbyshire, based on aggregating across all residents in all the LSOA zones.

Health and carbon impacts in Derbyshire

A modified version of the 2014 HEAT tool was used to calculate health impacts, based on the extra physical activity gained through new cycling trips in each scenario, minus the physical activity displaced in any former walking trips that have now been switched to cycling. CO2 impacts are derived from changes in distance driven per scenario.

Table 2 illustrates the health and carbon impacts of the different scenarios in Derbyshire, based on aggregating across all residents in all the LSOA zones.
