Data for

This page contains data downloads for , the region currently displayed on the ‘Map’ tab. These are available as CSV files containing the zone and line baseline and scenario data, and in geographic files that can be read by R (.Rds) or GIS programs such as QGIS (.geojson).

Please be aware that equivalent files for different geographies and different regions have the same file name when downloaded (e.g. ‘z.geojson’ for both the MSOA and LSOA regional zones files, for all regions). To avoid confusion you may want to rename those files or save them in labelled folders. Note also that when downloading 'csv' files using a Firefox browser, the file will open onscreen in a new webpage. If you then type "Ctrl + S" the onscreen data will download as a csv file.

Commuting data for lines

Download the Census 2011 and scenario data for lines in .

The ‘All flows’ files include all commuters who live or work in . Destinations are grouped as ‘Other’ for origin-destination pairs with a fastest-route distance >30km, or with a destination outside England or Wales. In addition, for the LSOA ‘All flows’ file, destinations are grouped as ‘Under 3’ for origin-destination pairs where the bidirectional number of commuters (by any mode) would be 1 or 2. This is done for data protection reasons.

The ‘Straight lines’, ‘Fast route’ and ‘Quieter routes’ files only include between-zone lines with fast route distance under 20km. In addition, the minimum number of commuters (by any mode, in either direction) must be 10 (MSOA) or 5 (LSOA).

The MSOA route network was calculated using within-region between-zone flows containing a minimum number of commuters (usually 10, but higher in large regions - see ‘Region stats’ tab). The LSOA route network includes all between-zone LSOA flows with fast route distance under 20km (i.e. including inter-regional flows and with no minimum number of commuters).
