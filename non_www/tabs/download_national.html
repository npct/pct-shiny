National Data

This page contains data downloads for the whole of England and Wales. These are available as CSV files containing the zone and line baseline and scenario data, and in geographic files that can be read by R (.Rds) or GIS programs such as QGIS (.geojson). Very large geographic files are available in R format only.

Please be aware that equivalent files for different geographies have the same file name when downloaded (e.g. ‘z_all.geojson’ for both the MSOA and LSOA national zones files). To avoid confusion you may want to rename those files or save them in labelled folders. Note also that when downloading 'csv' files using a Firefox browser, the file will open onscreen in a new webpage. If you then type "Ctrl + S" the onscreen data will download as a csv file.

Commuting data for zones

Download the Census 2011 and scenario attribute data for all zones in England and Wales.

  • Zones (local authorities)*: csv Rds geojson - Codebook
  • Zones (MSOA): csv Rds geojson - Codebook
  • Zones (LSOA):csv Rds geojson - Codebook

* Data aggregated from the LSOA zones file. Note that different units are used for the health economic savings (million pounds not pounds) and carbon savings (tonnes not kilograms of CO2e).

Commuting data for flows

Download the Census 2011 and scenario data for all flows starting or ending in England or Wales.

In the ‘All flows’ files, destinations are grouped as ‘Other’ for origin-destination pairs with a fastest-route distance >30km, or with a destination outside England or Wales. In addition, for the LSOA ‘All flows’ file, destinations are grouped as ‘Under 3’ for origin-destination pairs where the bidirectional number of commuters (by any mode) would be 1 or 2. This is done for data protection reasons.

The ‘Straight lines’, ‘Fast route’ and ‘Quieter routes’ files only include between-zone flows with fast route distance under 20km. In addition, the minimum number of commuters (by any mode, in either direction) must be 10 (MSOA) or 5 (LSOA).

The LSOA route network includes all between-zone LSOA flows with fast route distance under 20km (with no minimum number of commuters).
