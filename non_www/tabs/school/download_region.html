Data for - school layer

This page contains data downloads for the schools layer for , the region currently displayed on the ‘Map’ tab. These are available as CSV files containing baseline and scenario attribute data for zones and schools, and as geographic files that can be read by R (.Rds) or GIS programs such as QGIS (.geojson).

Please be aware that equivalent files for different purposes, geographies or regions have the same file name when downloaded (e.g. 'z.geojson' for all regions). To avoid confusion you may want to rename those files or save them in labelled folders. Note also that when downloading 'csv' files using a Firefox browser, the file will open onscreen in a new webpage. If you then type "Ctrl + S" the onscreen data will download as a csv file.

School travel data for zones

Download anonymised National School Census 2011 baseline data and scenario attribute data for all children who live in . These files present aggregate data for all children living in a given zone.

Baseline and scenario data are not presented for cells containing 1 or 2 children. Instead these data are set as missing ('NA').

  • Zones (LSOA): csv Rds geojson - Codebook

School travel data for schools

Download anonymised National School Census 2011 baseline data and scenario data for schools in . These files present aggregate data for all children attending a given school.

Baseline and scenario data are not presented for cells containing between 1 and 5 children. Instead these data are set as missing ('NA').

  • Schools: csv Rds geojson - Codebook

School travel data for the route network

Download anonymised National School Census 2011 baseline and scenario cycling estimates for the route network in . This was calculated using all fast routes, but only presents segments containing at least 10 cyclists in the Go Dutch scenario (in London, this threshold is set set to 50 for the web application due to performance considerations - see the 'Region stats' tab).

Baseline and scenario data are not presented for route segments containing 1 or 2 cyclists in the NSC 2011. Instead these data are set as missing ('NA').

  • Route network (LSOA, clickable): Rds geojson - Codebook
