National Data - school layer

This page contains data downloads for the schools layer for the whole of England. These are available as CSV files containing baseline and scenario attribute data for zones and schools, and as geographic files that can be read by R (.Rds) or GIS programs such as QGIS (.geojson).

Please be aware that equivalent files for different purposes and geographies have the same file name when downloaded (e.g. ‘z_all.geojson’ for national zones files for both commuting and school travel). To avoid confusion you may want to rename those files or save them in labelled folders. Note also that when downloading 'csv' files using a Firefox browser, the file will open onscreen in a new webpage. If you then type "Ctrl + S" the onscreen data will download as a csv file.

School travel data for zones

Download anonymised National School Census 2011 baseline data and scenario attribute data for all zones in England. These files present aggregate data for all children living in a given zone.

  • Zones (local authorities)*: csv Rds geojson - Codebook
  • Zones (LSOA): csv Rds geojson - Codebook

*Note that different units are used for the carbon savings (tonnes not kilograms of CO2e).

School travel data for schools

Download anonymised National School Census 2011 baseline data and scenario data for schools in England. These files present aggregate data for all children attending a given school.

  • Schools: csv Rds geojson - Codebook

Please see the codebooks for a description of the variables included in these download files, and the User Manual for further details of how these variable were calculated. You can also contact us at pct@pct.bike.
