Data for - commuting layer

This page contains data downloads for the commuting layer for , the region currently displayed on the ‘Map’ tab. These are available as CSV files containing baseline and scenario attribute data for zones and lines, and as geographic files that can be read by R (.Rds) or GIS programs such as QGIS (.geojson).

Please be aware that equivalent files for different purposes, geographies or regions have the same file name when downloaded (e.g. ‘z.geojson’ for both the MSOA and LSOA regional zones files, for all regions). To avoid confusion you may want to rename those files or save them in labelled folders. Note also that when downloading 'csv' files using a Firefox browser, the file will open onscreen in a new webpage. If you then type "Ctrl + S" the onscreen data will download as a csv file.

Commuting data for flows

Download the Census 2011 and scenario data for flows in .

The ‘All flows’ files include all commuters who live or work in . Destinations are grouped as ‘Other’ for origin-destination pairs with a fastest-route distance >30km, or with a destination outside England or Wales. In addition, for the LSOA ‘All flows’ file, destinations are grouped as ‘Under 3’ for origin-destination pairs where the bidirectional number of commuters (by any mode) would be 1 or 2. This is done for data protection reasons.

The ‘Straight lines’, ‘Fast route’ and ‘Quieter routes’ files only include between-zone flows with fast route distance under 20km. In addition, the minimum number of commuters (by any mode, in either direction) must be 10 (MSOA) or 5 (LSOA).

The LSOA route network includes all between-zone LSOA flows (i.e. including inter-regional flows), with no minimum number of commuters.
